Kolea is a fictional airline for which I created a brand for. This includes brainstorming the colors, typography and user flow. All of this based is on the target audience, research and interests. My main focus with this project was to create a streamline experience with booking a flight and checking your seat because those that are flying are probably already having a hard time.
